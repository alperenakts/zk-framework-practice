# ZK Framework Practice

This repository contains small practice applications developed to understand
the basics of the ZK Framework and its MVVM architecture.

## Technologies
- Java 8
- ZK Framework 6.5.2
- Maven
- Jetty

## Mini Applications

### 1. Mini Counter
A simple counter application to demonstrate:
- ZUL UI binding
- ViewModel usage
- @Command and @NotifyChange annotations

### 2. User & Role Management
A small application that demonstrates:
- MVVM pattern
- Service and Repository layers
- Duplicate user-role validation
- Confirmation dialogs
- Role-based user listing

## Project Structure

```text
src/main/java
└── com/alperen/zkmini
    ├── vm
    ├── service
    ├── repository
    └── domain
```
## Purpose
The purpose of this project is to practice ZK Framework concepts and gain
hands-on experience with MVVM-based UI development.
